Supervised learning and universal function approximation

    More or less the same thing

    Optimization of functionals very well studied field

    Extremely useful and important technique for about any quantitative stuff

    Probably not a very good model of behavior as is

Optimization AKA "Programming"

    Given y = f(x), find x that produces minimal (or maximal) y.

    Very simple, but very abstract, concept.

    E.g. y is height, f gives height at position x. What is the the lowest position?

    f(x) often known as "loss" when minimizing

Approximation function

    "Assume a parabolic crater" f(x) = ax², i.e. y = ax²

    Typically smooth so it is easy to "climb"

    In some special cases convex, meaning we can find the actual bottom! Rare in practice.

    LOTS of methods for solving this. Gradient descent is the most popular ML method. Lots of tricks!

Function approximation

    Can't just assume a "spherical cow" or "parabolic crater" always.

    Get some very flexible function, like lots of simple functions added together!

    It's a (very) artifical neural network!

    Finding flexible-enough but not too flexible function crucial. Aka "network architecture".

ANN architectures

    Typically classification, just another kind of loss function (e.g. softmax)

    "Deep" not usually really that deep

    "Convolution" means that some "connections" are shared

    "Skip connection" means that connection jumps layers

    "Recurrent" means usually some sort of memory, gets really tricky!
